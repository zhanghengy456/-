邓伦，1992年10月21日出生于河北省石家庄市，中国内地影视男演员，毕业于上海戏剧学院表演系。

2012年，出演个人首部电视剧《花非花雾非雾》，从而正式进入演艺圈。

2013年，出演民国情感剧《新京华烟云》 。

2014年，主演都市情感轻喜剧《爱情上上签》。

2015年，在都市青春偶像剧《十五年等待候鸟》中饰演柳千仁 ；同年，出演年代剧《白鹿原》 。

2016年，在都市情感励志剧《因为遇见你》中饰演男主角李云恺，从而受到更多关注 ；同年，在都市励志情感剧《欢乐颂2》中饰演谢童 ；在《楚乔传》中饰演大梁太子萧策；随后，获得“腾讯视频星光大赏”年度青春大势艺人奖 。

2017年，相继主演了民国情感剧《海棠经雨胭脂透》 、古装神话剧《香蜜沉沉烬如霜》、都市奇幻剧《一千零一夜》 ；同年，作为固定嘉宾参加芒果TV亲子户外真人秀节目《爸爸去哪儿第五季》的录制。

2018年，获得国剧盛典年度新晋男演员奖 ；同年，作为常驻嘉宾参加湖南卫视情境类益智互动推理秀《我是大侦探》 。
